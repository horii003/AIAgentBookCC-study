## 1. 目的・適用範囲

### 1.1 目的
- AIエージェントが業務で利用される際の  
  **誤操作・誤判断・逸脱・リスクを防止**するための制御要件を定義する
- 入力・出力・判断・実行の各段階にガードレールを設ける

### 1.2 適用範囲
- 対象エージェント：
- 対象業務／ユースケース：
- 対象成果物：
- 適用フェーズ：会話／生成／実行

## 2. ガードレール全体構成（概要）

```text
[ユーザー入力]
   ↓（入力ガード）
[会話・判断]
   ↓（ルール検証）
[生成・実行]
   ↓（出力ガード）
[成果物／API実行]
```
